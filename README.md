# Dothem

**Dothem** is a simple task‑management app that helps you organize your day by adding tasks, tracking their status, and managing priorities. All your data is stored locally, with local login.

## Short Description

A lightweight to‑do application that lets you add tasks, set their importance, check their status, and sort them easily.

## Requirements

* **Task Registration**: Add new tasks with basic details.
* **Task Status**: Each task can be in progress, completed, or pending.
* **Task Importance**: Every task has an importance level (low / medium / high).
* **Local Login**: Ability to log in locally.
* **Database Storage**: All tasks are saved on the database.

## Additional Features

* Clean and simple interface.
* Easy task editing and deletion.
